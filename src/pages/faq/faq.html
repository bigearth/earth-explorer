 What blockchain is this? This is a Bitcoin Cash block explorer. What do the icons on each page mean?

Blocks

Block height

Total value in BCC of all the transactions included in this block.

Total number of transactions included in this block.

Number of blocks that have been processed since the previous block (including the block itself).

The network difficulty for finding the block hash.

Size of the block in bytes.

Indicates if the block is in the main chain or is an orphan.

Block hash.

Top of the Merkle tree, which can be used to verify that a given transaction was included in the block.

Name of the mining pool detected to be the creator of the block.

Hash of the transaction.

Total estimated value of the transaction in Satoshi.

Total number of blocks that have been processed since the transaction, including the block containing the transaction.

Total value of all the inputs in Satoshi.

Total value of all the outputs in Satoshi.

Total fees paid to the miner in Satoshi.

Size of the transaction in bytes.

Indicates if the transaction is a coinbase transaction.
